# biotech-investor-app

AI-powered biotech and pharma investor literacy tool. Enter any ticker 
to get a plain-English breakdown of a company's pipeline, clinical data, 
approval outlook, and downside risk. Built for retail investors who want 
to understand what they own before they buy.

## Features
- Company breakdown across 5 sections: what they do, pipeline stage, clinical data, case for approval, and downside risk
- Comparison mode between two companies in the same therapeutic area
- Integrated glossary of biotech terms

## Tech Stack
- Next.js
- Anthropic Claude API
- Deployed on Vercel
